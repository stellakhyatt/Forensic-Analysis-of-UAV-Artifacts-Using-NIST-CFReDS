## Project Overview
This project examines unmanned aerial vehicle (UAV) artifacts using the NIST CFReDS drone dataset.  The focus is on identifying and reviewing flight log artifacts and related files to help reconstruct drone activity using basic forensic tools.

## Dataset
NIST CFReDS Drone Dataset

## Tools Used
- FTK Imager
- CsvView

## Project Focus
- Identifying UAV-related files in the dataset
- Examine flight log data
- Review timestamps, GPS-related values, and telemetry-style fields where available
- Document findings with screenshots and notes

- 'screenshots/' - screenshots from FTK Imager and CsvView
- 'report/' - final report materials
